Recast An Analysis Reinterpretation Framework

Introduction

This is an early prototype for the RECAST framework. This web application communicates with the RECAST front-end at http://recast.perimeterinstitute.ca and provides a plugin model for analyses.

Instructions

  1. To test the RECAST service, click on the All Analyses link in the navigation above. Select the analyses that you want to recast. Alternatively you can also create a request on the main RECAST website (currently the development instance)
  2. To create a new request click the "New RECAST Request" button and fill out the form. After you created the request you can click through to the page describing your new request
  3. On the request page you can now upload simulated events for specific parameter points in the Les Houches format (currently they need a '*.events' extension as per ATLAS convention). To do that click the "Add Parameter Point" button, describe the parameter point and select the necessary event files (you don't need to worry about the cross-section and number of events for now. Dummy values are OK.)
  4. Now that you have created a parameter point, you can process the request by hitting the "Process" button. Once the results become available you can view the results for the parameter point (you will need to reload the page at the moment).
  5. Once the results have arrived, you can upload the response to the RECAST frontend by clicking the "Upload to RECAST" button
